# WDD330-Project-Yoga-Institute
# MOOD

The project would be a Yoga Institute platform. Users would be able to login, search classes, instructors and book the classes. May be processing payments if possible. The Administrator of the Yoga institute will be able to access students and instructors info. Get classes scheduled and payments, and commissions for each instructor depending on how many students attended. Solving the attendance and book process for students, allowing the administrators to see the incomes and classes. 

Using JS, HTML, CSS and Google APIs to authentication and Calendar.
